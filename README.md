# Webiste Personal Advokat

Website ini merupakan website profesional untuk seorang Advokat dan Konsultan Hukum, yang bertujuan memberikan informasi yang jelas, ringkas, dan terpercaya kepada calon klien. Website ini menampilkan berbagai informasi penting mengenai layanan hukum, latar belakang profesional, pendidikan, serta cara untuk menghubungi advokat secara langsung.

Proyek ini dibangun dengan fokus pada kesederhanaan, dan kemudahan navigasi, sehingga visitor dapat dengan cepat menemukan informasi yang mereka butuhkan. Selain itu, desain website dibuat responsif agar tampil optimal di berbagai perangkat, mulai dari desktop hingga mobile.

Website ini terdiri dari tiga halaman utama:

Home — Menampilkan pengenalan singkat serta tombol call-to-action untuk menghubungi advokat melalui WhatsApp.

Contact — Menyediakan informasi kontak dan akses cepat untuk berkomunikasi dengan advokat.

About — Berisi profil lengkap, riwayat pendidikan, serta perjalanan karier mulai dari kepolisian hingga menjadi seorang advokat profesional.

Website ini diharapkan dapat menjadi sarana efektif untuk memperkuat kredibilitas advokat sekaligus mempermudah klien dalam mendapatkan bantuan hukum.

## Teknologi yang digunakan

* Vite

Vite merupakan front-end tooling yang digunakan untuk mempermudah dan mempercepat dalam pengembangan website. Vite mendukung berbagai macam framework JavaScript, seperti React, Vue, Svelte, dll. Dan juga framework CSS, seperti Tailwind.

* HTML 

Hyper-text markup language atau HTML adalah sebuah markup language yang digunakan untuk membuat struktur sebuah website.

* CSS

Cascading style sheet atau CSS merupakan style sheet yang digunakan untuk mengubah tampilan dari markup language salah satunya adalah HTML.

* CSS Framework (Simple.css)

Simple.css merupakan framework CSS. Simple.css memberikan tampilan yang sederhana, tetapi terlihat bagus dan moder, selain itu memberikan tampilan yang responsif di berbagai perangkat, mulai dari desktop hingga mobile.

* TypeScript

TypeScript merupakan bahasa pemrograman yang typed-language, di-compile, Object-Oritented Programming, Prosedural, dll.
